# Japanese-Web-Game
### Purpose
Make learning Japanese more fun :D 

![alt text](https://media.giphy.com/media/ufszmdQvbMNH2/giphy.gif)
### How to play ?
- Go to this website https://jpcg.neocities.org/
- Just click on card to flip and find the equivalent pairs
### Word pairs
- Apple --- アップル
- Bus --- バス
- Teacher --- 先生
- Juice --- ジュース
